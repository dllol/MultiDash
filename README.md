# MultiDash
MultiDash is a tool that allows you to play the popular game Geometry Dash along with your friends!

MultiDash works by connecting you to a server and overlaying each player's movement on your screen, such that you can both see yourselves and play together.

# Installation
Due to a bug in batch files, we cannot fully automate this process, as such you need to follow these instructions:

- 1. Download the latest Node.JS version
- 2. Download .NET Core 3.1
- 3. Run the following commands in a  command promt:
```
npm i qs axios process
npm i -g electron
```
- 4. Finally, you can run `install.bat` and the app should start up.
